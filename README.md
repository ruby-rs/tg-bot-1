# tg-bot-1

Telegram-бот — личный трекер жизни: задачи и прогресс по категориям (работа, спорт, питание, стройка, машина, ипотека, отношения, отпуск), учёт веса и расходов, советы по питанию для набора массы.

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env   # укажи BOT_TOKEN
python -m bot.main
```

## Команды

- `/start` — регистрация, список команд
- `/panel` — панель прогресса по всем категориям (MarkdownV2)
- `/add` — добавить задачу в категорию
- `/tasks` — список задач с отметкой выполнения / удалением
- `/weight <кг>` — записать вес
- `/weightstats` — динамика веса
- `/expense` — записать расход по категории (для категории "Машина" — отдельный сценарий: заправка → литры → стоимость → способ оплаты)
- `/expenses` — расходы за текущий месяц
- `/habits` — отметить привычки за сегодня (✅ выполнено / ❌ пропущено / ➖ не отмечено, тап переключает статус)
- `/habitstats` — сколько раз выполнена каждая привычка в текущем месяце
- `/calendar` — календарь по месяцам: выбери месяц (◀️/▶️), затем день, чтобы отметить привычки и записать заправку задним числом; дни с выполненными привычками помечены ✅
- `/tips` — советы по питанию и набору массы
- `/cancel` — отменить текущее добавление задачи/расхода
- `/help` — показать список команд ещё раз

Данные хранятся в SQLite (`life_tracker.db` по умолчанию, путь настраивается через `DB_PATH`).

Бот держит одно "рабочее" сообщение на чат и переиспользует его для панели, списков задач, отчётов и трекера привычек — команды и кнопки редактируют это сообщение вместо того, чтобы плодить новые.

Список команд с описаниями регистрируется в Telegram при старте (`set_my_commands`), поэтому они видны в меню бота («/» рядом с полем ввода) с подсказками.

Панель (`/panel`) — компактная: прогресс по категориям и быстрые кнопки (➕ Задача, 💸 Расход, 📋 Категории, 📅 Календарь, ✅ Привычки, 📊 Расходы, ⚖️ Вес, 🍗 Советы). Полный список категорий вынесен в отдельный экран за кнопкой "📋 Категории", чтобы не загромождать панель девятью кнопками.
